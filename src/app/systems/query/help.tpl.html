Query syntax help

The basic syntax is composed of prefix.key with =/!= some value for example: 

 
   owner=foo env!=prod machine.cpus=4
The query is performed on the raw system model json structure, to view json example the Swagger API can be used.

In order to fetch all systems a star query can be used: 


    *

Note that the query is encoded in the url, so you can share them and move back and forward between them.
